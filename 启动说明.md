# Redis可视化管理工具 - 快速启动指南

### 1. 安装依赖
```bash
cd /Users/admin/code/redisTools
pip3 install redis --break-system-packages
```

### 2. 运行程序

**方式一：直接运行**
```bash
python3 main.py
```

**方式二：使用启动脚本**
```bash
./run.sh
```

## 🎯 使用流程

### 第一次使用

1. **启动程序**
   ```bash
   python3 main.py
   ```

2. **添加Redis连接**
   - 点击顶部 **"管理连接"** 按钮
   - 点击 **"添加"** 按钮
   - 填写连接信息：
     - 别名：例如 `本地开发`、`生产环境`
     - 主机：`127.0.0.1`（或服务器IP）
     - 端口：`6379`（默认端口）
     - 密码：如果有密码就填写，否则留空
     - 数据库：`0`（默认）
   - 点击 **"测试连接"** 验证
   - 关闭窗口保存

3. **连接Redis**
   - 程序会自动连接（或提示你连接）
   - 连接成功后，状态栏显示绿色

4. **开始管理Key**
   - 左侧：快捷操作按钮
   - 中间：Key列表（支持搜索、分页）
   - 右侧：Key详情查看

## 📋 主要功能

| 功能 | 操作位置 | 说明 |
|------|---------|------|
| 新增Key | 左侧"➕ 新增Key" | 支持5种数据类型 |
| 编辑Key | 选中Key后点击"✏️"或双击 | 修改Key的值 |
| 删除Key | 选中Key后点击"🗑️" | 需确认 |
| 重命名 | 选中Key后点击"📝" | 修改Key名称 |
| 设置过期 | 选中Key后点击"⏰" | 设置TTL（秒） |
| 搜索Key | 中间搜索框 | 支持模糊匹配 |
| 刷新列表 | 左侧"🔍 刷新" | 更新Key列表 |
| 清空数据库 | 左侧"⚠️"按钮 | 危险操作，需确认 |
| 服务器信息 | 顶部"服务器信息" | 查看Redis状态 |

## ❓ 常见问题

### Q1: 提示"未找到redis模块"
**解决**: 安装redis库
```bash
pip3 install redis --break-system-packages
```

### Q2: 连接失败
**检查**:
1. Redis服务是否启动：`redis-cli ping` 应返回 `PONG`
2. 主机端口是否正确
3. 密码是否正确
4. 防火墙是否阻止

### Q3: 如何启动本地Redis测试？
```bash
# 安装Redis（如果没有）
brew install redis

# 启动Redis
redis-server

# 或使用后台模式
redis-server --daemonize yes
```

### Q4: 连接配置保存在哪里？
保存在 `connections.json` 文件中（已加入.gitignore）

### Q5: 如何备份连接配置？
备份 `connections.json` 文件即可
